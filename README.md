# To Do List

Aplicação simples de lista de tarefas desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Remover tarefas

- Adicionar tarefas pressionando Enter
- Interface simples e intuitiva

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais do desenvolvimento web, como:

- Manipulação do DOM
- Eventos em JavaScript
- Criação dinâmica de elementos HTML
- Estruturação de projetos front-end
